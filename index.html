LobsterPages

Project of a webpage covering the exploits and advances of the team the Almighty Lobsters through the CanSat competition. This being the front-end part of the project (as of now).

Technologies used

  • React.js
  • React-router
  • Webpack
  • Babel
  • Sass
  • ESLint

How to run

git clone https://github.com/AlmightyLobsters/LobsterPages.git to clone repository
yarn / npm install to install packages
yarn start / npm start to spin up the hot-reloading webpack-dev-server
(you will find it on http://localhost:8080/ in your browser)
yarn run build / npm run build to build the project and get a static web page
