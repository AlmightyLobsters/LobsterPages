LobsterPages

Project of a webpage covering the exploits and advances of the team the Almighty Lobsters through the CanSat competition. This being the front-end part of the project (as of now).

Technologies used

  • React.js
  • React-router
  • Webpack
  • Babel
  • Sass
  • ESLint

How to run

yarn to install packages
yarn start to spin up the hot-reloading webpack-dev-server
yarn run build to build the project and get a static web page
